Factory Method Pattern(工厂方法模式)

  1. Define an interface for creating an object, but let subclasses decide which class to instantiate. Lets a class defer instaniation to subclasses.

  2. UML

Abstract Factory Pattern(抽象工厂模式)

  1. Provide an interface for creating families of related or dependsent objects without specifying their concrete class.
